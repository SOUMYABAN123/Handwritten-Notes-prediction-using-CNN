# Handwritten-Notes-prediction-using-CNN
Handwritten notes prediction using CNN and reduce the loss function and increase the model accuracy to 95% +
Dataset - https://www.kaggle.com/datasets/hojjatk/mnist-dataset
Activation Function-ReLU, softmax
Loss Function-Catagorical_crossentropy
OneHotEncode-to_catagorical()
Optimizer-Adam
Framework-Tensorflow, Keras
Library Used-numpy, pandas, matplotlib
